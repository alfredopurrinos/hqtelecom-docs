---
title: "What is the difference between the T-lock call blocker version 5.0 and version N2?"
sidebar_position: 44
format: md
---

In terms of features, they both have the same, but the version 5.0's software has fixed some compatibility issues with major carriers e.g. less susceptible to power issues and noise. It also has a new design, logo, and retail / gift box.
