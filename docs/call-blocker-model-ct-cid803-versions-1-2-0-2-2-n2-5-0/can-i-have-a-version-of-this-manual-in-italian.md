# Iniziare:  Il T-Lock è spedito dalla fabbrica pronto per iniziare a lavorare quando si è connessi a qualsiasi / DTMF linea telefonica compatibile FSK utilizzando la presa LINE. Una volta installato, è sufficiente attendere per iniziare a ricevere le chiamate, quindi utilizzare i tasti di scorrimento UP o DOWN (◄ o ►) per sfogliare l'elenco delle ultime chiamate in entrata. Durante la navigazione, è possibile premere il pulsante lista nera per aggiungere numeri indesiderati alla blacklist. E 'così semplice IMPORTANTE: Si prega di verificare che la linea telefonica è il servizio di identificazione del chiamante. Il tuo T-Lock non funziona senza di essa! Se non si è sicuri, contattare il proprio fornitore di servizi telefonici.  Come programmare il vostro T-Lock Call Blocker  PREPARAZIONE: Prima di poter iniziare a programmare il vostro T-Lock, è necessario disporre di un telefono (cablata o wireless) con una connessione dial-pad collegato alla presa "PHONE" sul retro della T-Lock (come mostrato nella installazione Series diagramma).  1.0 Impostazione dell'anno  Attenersi alla seguente procedura per modificare la 4a cifra rispetto all'anno (ad esempio da 201 2-201 3).  Fase 1. Premere il pulsante SET (display LCD Set 1234567)  Fase 2. Sollevare il ricevitore (1) e premere 1 utilizzando la manopola-pad (LCD visualizza 20 1 2 ° e 3 ° cifra lampeggia indicando che può essere modificato).  Di nuovo Fase 3. Premere il pulsante SET per passare dal 3 ° al 4 ° cifra dell'anno (201 2).  Fase 4. Premere il pulsante UP (◄) per cambiare la cifra lampeggiante (3,4,5).  Punto 5. Una volta visualizzata la cifra corretta, riagganciare il ricevitore, e attendere 15 secondi per uscire dalla modalità di programmazione.  1.1 Impostazione di data e ora  . Se il fornitore di servizi telefonici è FSK compatibile, data e ora sul vostro T-Lock verrà aggiornato automaticamente quando viene collegato dopo la prima chiamata in caso contrario, attenersi alla seguente procedura per impostare data e ora:  Dopo aver impostato ultima cifra dell'anno nella fase 4, premere il tasto SET di nuovo per cambiare il mese (MM), utilizzare dial-pad per inserire il mese corretto, quindi premere di nuovo SET per cambiare il giorno (DD), utilizzare dial-pad per inserire il mese corretto, quindi premere di nuovo SET per cambiare l'ora (HH), utilizzare dial-pad per inserire il mese corretto, quindi premere di nuovo SET per cambiare i minuti (MM). Poi, riagganciare il microtelefono e attendere 15 secondi per uscire dalla modalità di programmazione.  2.0 Impostazione dei livelli di luminosità del display LCD  Le impostazioni predefinite per la luminosità del display è 2. Per modificare le impostazioni, seguire le istruzioni riportate di seguito:  Fase 1. Premere il pulsante SET (display LCD Set 1234567)  Fase 2. Sollevare il ricevitore e premere 2 utilizzando il dial-pad. (display LCD "LCD 2")  Passaggio 3. Premere UP o DOWN (◄ o ►) per modificare i livelli LCD da 2 a 1,2,3,4  Fase 4. Al termine, riagganciare il microtelefono e attendere 15 secondi per uscire dalla modalità di programmazione.  3.0 Aggiunta di uscita (selezione) i numeri bloccati alla lista nera  Attenersi alla seguente procedura per aggiungere manualmente i numeri di telefono (anche prefissi o prefissi) che si desidera bloccare composizione dei numeri dalla linea telefonica.  IMPORTANTE: Assicurarsi che sia installato il T-Lock in serie con il telefono esistente (s) in modo che il T-Lock per bloccare tutti i numeri di chiamate in uscita.  Fase 1. Premere il pulsante SET (display LCD Set 1234567)  Fase 2. Sollevare il ricevitore (1) e premere 3 (display LCD visualizza un cursore lampeggiante "\_" indica la modalità di ingresso)  Punto 3. Immettere il numero di telefono (2) (o il prefisso o prefisso) che si desidera bloccare.  Passo 4. Premere DOWN (◄ o ►) UP o per salvare (4) (display LCD "successo").  Ripetere i passaggi 3-4 per proseguire inserendo numero (s) che si desidera bloccare.  Passo 5. Al termine, riagganciare il microtelefono e attendere 15 secondi per uscire dalla modalità di programmazione.  3.1 Rimozione di uscita (selezione) numeri bloccati dalla lista nera  Per rimuovere un numero dalla lista bloccata (3), è necessario premere SET, premere 3, utilizzare i tasti UP & DOWN (◄ o ►) per scorrere l'elenco dei numeri bloccati che si desidera rimuovere, quindi premere il pulsante REMOVEper rimuovere il numero visualizzato (LCD (3) viene visualizzato "DEL" e "successo").  4.0 Aggiunta di numeri di telefono in entrata alla lista nera  Seguire le istruzioni riportate di seguito per aggiungere i numeri di telefono in entrata alla lista nera.  Fase 1. Premere il pulsante SET (display LCD Set 1234567)  Fase 2. Sollevare il ricevitore e premere 4 (display LCD viene visualizzato il primo numero memorizzato nella lista nera. Se vuoto, un cursore lampeggiante deve essere visualizzato come "\_" indica la modalità di ingresso)  Passo numero di telefono 3. Inserire (2) da aggiungere alla lista nera in arrivo  Passo 4. Premere DOWN (◄ o ►) UP o per salvare (4) (display LCD "successo").  Ripetere i passaggi 2-4 per aggiungere fino a 1.200 numeri (fino a 16 caratteri di lunghezza, ciascuno)  Passo 5. Al termine, riagganciare il microtelefono e attendere 15 secondi per uscire dalla modalità di programmazione.  Consultare la Sezione 10.2 per rimuovere i numeri di telefono in entrata da lista nera.  5.0 Aggiunta di codici Area in arrivo o prefissi nella lista nera  Seguire le istruzioni riportate di seguito per aggiungere codici e prefissi nella lista nera della zona telefono in entrata.  Fase 1. Premere il pulsante SET (display LCD Set 1234567)  Fase 2. Sollevare il ricevitore e premere 5 (Se tua lista nera è vuota, un cursore lampeggiante deve essere visualizzato come "\_" indica la modalità di ingresso. In caso contrario, il display LCD visualizza il primo numero memorizzato nella lista nera)  Fase 3. Inserire il prefisso o il prefisso che si desidera bloccare (2) (esempio prefisso: 305, il prefisso 305-558) utilizzando la manopola-pad sul vostro apparecchio telefonico.  Fase 4. Premere UP o DOWN (◄ o ►) per salvare il numero (4) (LCD visualizza "SAVE" e "successo"). Ripetere i passaggi 2 e 3 per aggiungere fino a 10 gruppi (fino a 10 caratteri di lunghezza, ciascuna)  Passo 5. Al termine, riagganciare il microtelefono e attendere 15 secondi per uscire dalla modalità di programmazione.  5.1 Rimozione Prefissi in entrata o prefissi dalla Blacklist  Seguire i passaggi 1 e 2 di cui sopra, quindi seguire questi passaggi aggiuntivi:  Fase 3. Utilizzare i pulsanti SU o GIÙ (◄ o ►) per l'elenco dei numeri bloccati sfogliare  Fase 4. Quando si visualizza il numero che si desidera rimuovere, premere REMOVE (LCD (3) viene visualizzato "DEL").  Passo 5. Al termine riagganciare il ricevitore, e attendere 15 secondi per uscire dalla modalità di programmazione.  6.0 Accendere / OFF funzione per bloccare tutte le chiamate in arrivo senza ID chiamante Info  Questa funzione è disattivata (OFF) come impostazione predefinita di fabbrica. Si prega di seguire le istruzioni riportate di seguito se si desidera disattivare questa funzione su "ON" e bloccare tutte le chiamate senza informazioni ID del chiamante.  Fase 1. Premere il pulsante SET (display LCD Set 1234567)  Fase 2. Sollevare il ricevitore e premere 6 (display LCD visualizza "OFF")  Passo 3. Premere DOWN (◄ o ►) UP o per alternare funzione "ON" e "OFF".  Fase 4. Quando viene visualizzato "ON", riagganciare il microtelefono e attendere 15 secondi per uscire.  7.0 Modifica del codice d'accesso  Fabbrica di codice di accesso di default del T-Lock è 0000. Seguire le istruzioni riportate di seguito per cambiare è:  Fase 1. Premere il pulsante SET (display LCD Set 1234567)  Fase 2. Sollevare il ricevitore e premere 7 (display LCD "INPUT-")  Fase 3. Inserire il vecchio codice pass (default 0000) e premere il tasto SET (display LCD "N-PASSWORD")  Fase 4. Inserire il nuovo codice di accesso (fino a 4 cifre), premere il tasto SET  Fase 5. Immettere nuovamente nuovo codice di accesso e premere il tasto SET per salvare (4) (display LCD visualizza "avere successo")  Passo 6. Riagganciare microtelefono e attendere 15 secondi per consentire T-Lock per uscire dalla modalità di programmazione.  Nota: è possibile utilizzare "8289", se si dimentica il codice di accesso per ripristinare di nuovo al valore predefinito (0000).  8.0 Sfoglia entrata numeri di telefono  È possibile premere i tasti UP o DOWN (◄ o ►) per visualizzare i record delle chiamate in entrata recenti (ultime 100 chiamate ricevute). Il display LCD visualizza il numero di telefono (fino a 16 cifre), la data e l'ora di ogni chiamata.  9.0 Sfoglia uscita Numeri di telefono  Premere OUT per navigare recenti numeri di telefono in uscita effettuate dal telefono (s) collegati alla presa PHONE del T-Lock. DOWN (◄ o ►) Premere UP o per scorrere l'elenco. LCD visualizza "NO CODE 'quando non ci sono le chiamate. Ogni record può visualizzare fino a 16 cifre.  10.0 Rimozione Numeri da elenchi (INCOMING, OUT, e Blacklist)  10.1 Rimozione di numeri telefonici con lista dei messaggi entranti  Per rimuovere i numeri dall'elenco INCOMING delle chiamate recenti (T-Lock può contenere fino a 100 record, 16 cifre ciascuno), è sufficiente utilizzare i tasti UP o DOWN (◄ o ►) per passare in rassegna la lista. Quindi, premere ilpulsante Rimuovi per eliminare il numero dalla lista (LCD (3) viene visualizzato "DEL"). Per rimuovere tutto l'elenco dei record di chiamata in arrivo, premere il tasto REMOVE (delicatamente) per più di 3 secondi.  10.2 Rimozione di numeri di telefono dalla lista nera  Per rimuovere i numeri dalla memoria Blacklist (fino a 1.200 record), premere il pulsante blacklist, quindi utilizzare UP o DOWN (◄ o ►) per sfogliare l'elenco. Quindi, premere il pulsante Rimuovi per eliminare il numero dalla lista (LCD (3) viene visualizzato "DEL"). Per rimuovere tutto l'elenco, premere il pulsante REMOVE più di 3 secondi. NOTA: È necessario inserire codice di accesso (di default 0000) seguito dal tasto REMOVE quando display LCD "PASSWORD". Si tratta di una misura di sicurezza per evitare che elimina l'elenco involontariamente.  10.3 Rimozione di numeri di telefono dall'elenco OUT  Per rimuovere i numeri dalla lista in uscita dei numeri di telefono più recenti (fino a 50 record) premere il pulsante blacklist, quindi utilizzare i tasti UP o DOWN (◄ o ►) per passare in rassegna la lista. Ora, è possibile premere ilpulsante Rimuovi per eliminare qualsiasi numero di telefono che si desidera (display LCD "successo"). Per rimuovere l'intero elenco, premere il pulsante REMOVE più di 3 secondi.  11.0 Blacklist Funzionamento  T-Lock si disconnette automaticamente (off-hook e terminare la chiamata) se il numero del chiamante è memorizzato nella lista nera. Si può sentire 0-2 squilli prima che le chiamate siano scollegati.  11.1 Aggiunta di nuovi numeri di telefono nella lista nera:  Ci sono 2 modi per aggiungere i numeri di telefono indesiderati alla blacklist:  (1) Quando una chiamata indesiderata squilla ... Basta premere il pulsante Blacklist mentre il telefono sta squillando (prima di rispondere) per scollegare la chiamata indesiderata e aggiungere il numero alla lista nera.  (2) Dopo aver risposto a una chiamata indesiderata ... È possibile utilizzare lo scorrimento UP / DOWN (◄ o ►) per sfogliare l'elenco delle ultime chiamate in entrata, quindi premere il pulsante lista nera per aggiungere il numero di telefono indesiderati per elencare (display LCD " SAVE "e" successo ").  12.0 BLOCCO Caratteristica  La funzione "LOCK" è impostato su "OFF" come impostazione predefinita di fabbrica. Si può lift-off ricevitore e l'ingresso \* 000111 # per attivare "ON" la funzione di blocco e bloccare tutte le funzioni (display LCD "LOCK"). Se il display LCD del vostro T-Lock "LOCK", quindi ricevitore e l'ingresso \* 123456 # lift-off per attivare la funzione di blocco "OFF", e abilitare le funzioni di tutte le T-Lock.  13.0 Display LCD Avvisi e messaggi  Il display LCD visualizza i seguenti messaggi o avvisi.  L'allarme 'NUOVO' indica che il numero di telefono è nuovo (ad esempio, recentemente aggiunto alla memoria del T-Lock)  L'allarme 'REP' indica che il numero di telefono è già in memoria.  L'allarme 'MAIL' indica che si dispone di un nuovo messaggio in segreteria (questa funzione è fornita dal gestore di telefonia come parte del servizio ID chiamante).  Il display visualizza anche la data corrente di settimana: Dom, Lun, Mar, Mer, Gio, Ven, SAT.  14.0 Note di programmazione (x)  (1) Quando si utilizza il portatile per programmare la tua T-Lock, si prega di ignorare i messaggi vocali (o eventuali errori di linea acustici) si può sentire sulla linea telefonica.  (2) In modalità di immissione, è possibile premere il pulsante OUT in qualsiasi momento per correggere i numeri.  (3) Quando si rimuove un numero, il display può mostrare un cursore lampeggiante "\_" - Questo è normale!  (4) Questo dispositivo utilizza una tecnologia (chiamato EEPROM), che permette di conservare tutta la programmazione e numeri memorizzati anche quando scollegato dalla linea telefonica.  Garanzia e assistenza  Hi Q Telecom Inc. garantisce i propri prodotti sono esenti da difetti di materiale e di fabbricazione per un periodo di un (1) anno dalla data di acquisto. Tutti gli articoli da riparare in garanzia sono da inviare, insieme con l'originale scontrino a: HQ TELECOM, 16562 NW 83 PL, HIALEAH FL 33016  Assistenza per il prodotto  Si prega di visitare il nostro sito all'indirizzo http://hqtelecom.com per trovare le ultime informazioni su questo prodotto, comprese le domande frequenti. È possibile utilizzare il nostro modulo di contatto on-line di porre domande, e fornire un feedback e / o suggerimenti. Inoltre, il nostro staff di supporto è disponibile via telefono al numero 1-786-221-5997 (Lunedi a Venerdì, 11:00-07:00 EST).  Questo manuale è disponibile anche online in altre lingue (olandese, cinese, francese, tedesco, italiano, spagnolo, russo, e altri).
