sidebar_position: 10

No. This device is plug-and-play! There are 2 ways of installing it; series or parallel. Please [watch this YouTube Video](https://www.youtube.com/watch?v=YN9U6QUZ_qk) to see how easy it is to setup (<https://www.youtube.com/watch?v=YN9U6QUZ_qk>)  
NOTE: The year setting is the only item that requires manual setting, however it's optional and not required for this device to work properly.
