Yes, you can mute the instruct the dialer to mute when it dials so the user can not hear the tones when it dials out via the register 13.  
  
Register 13 Mute dialing  
Valid Entries:  
0 = No mute 1 = Mute  
  
Scenario: You want the dialer to mute the tones when dialing out. How to:  
Step 1) Enter **# 0 \*** -- waif for beep  
Step 2) **13** -- wait for beep  
Step 3) **1**  -- waif for beep  
Step 4) **##** -- Wait for 2 beeps
