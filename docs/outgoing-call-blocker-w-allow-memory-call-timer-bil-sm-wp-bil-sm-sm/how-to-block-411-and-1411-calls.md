Please
reset the unit following feature # 7 in the manual
(<https://cdn.shopify.com/s/files/1/0624/0131/4971/files/T-Lock-Manual-Bil-TCR.pdf?v=1782255190>).    
  
If you can program your call blocker to block all calls that start with 1 or 4 (simplest programming), then
program feature # 5 to 01.   At this time, you should not be able to make any outgoing calls that require more
than 1 digit.  Then, you should add the prefixes that you want to allow using feature # 4
(code 1034).  We recommend adding:  
skip 1  
2#  
3#  
skip 4  
5#  
6#  
7#  
8#  
9#  
Remember you can always bypass restrictions by entering 102222.  
  
If you NEED to allow calls that start with 1 (except 14xx) and 4 (except 41x),
then follow the steps below:  
  
Program feature # 5 to 02.  At this time,
you should not be able to make any calls that require dialing a number that is
longer than 2 digits e.g. 411, etc.  After you confirm that you are not able to make any outgoing calls to 411 or 1411,
then you may want to add the prefixes you like to allow using feature # 4.  
We would recommend adding:    
2# -- to allow all calls that start with 2  
3# -- to allow all calls that start with 3  
5# -- to allow all calls that start with 5  
6# -- to allow all calls that start with 6  
7# -- to allow all calls that start with 7  
8# -- to allow all calls that start with 8  
9# -- to allow all calls that start with 9  
  
If you wish to allow calls that start with 40,42,43,44,45,46,47,48 or 49, then add these:   
40#   
Skip 41 since you want to block 411  
42#  
43#  
44#  
45#  
46#  
47#  
48#  
49#  
  
If you NEED to allow calls that start with 11,12,13,15, 16,17,18, or 19 (except 14), then add:  
11#  
12#  
13#  
Skip 14  
15#  
16#  
17#  
18#  
19#
