---
title: "Can I add a delay prior to the number being auto dialed out?"
sidebar_position: 7
format: md
---

Yes,  please follow these programming steps:  
  
**\*# 1234**- entering re-programming mode   
**\*\*\***(beep) **-**entering instruction set **\*\* 22 XY \***(beep)- programming register to add delay prior to auto dial number  
**Hangup**

Where   
XY = 10 adds 1 second  
XY = 20 adds 2 seconds  
XY = 30 adds 3 seconds

etc
