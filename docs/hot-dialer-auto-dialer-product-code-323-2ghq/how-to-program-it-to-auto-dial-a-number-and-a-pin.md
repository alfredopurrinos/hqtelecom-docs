Below are the instructions:

\*\*\*  [wait for beep]  
\*\*9\*\*  [wait for beep]    
\*9\*1\*\*200\*NUMBER\*PIN\*\*  [wait for beep -- programs number and pin to be auto dialed]

\*5\*1\*0050\*\*  [wait for beep -- adds a 5 second delay between number and pin]

\*\*235\*  [wait for beep -- enables off-hook dialing]
