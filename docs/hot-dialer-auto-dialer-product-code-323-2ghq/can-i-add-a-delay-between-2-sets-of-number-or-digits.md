---
title: "Can I add a delay between 2 sets of number or digits?"
sidebar_position: 4
format: md
---

Yes, you can! Please follow the instructions below to remote program (remove first line if local programming):  
  
**\* # 1234** (wait for beep)

**\* \* \*** (wait for beep)     

**\* \* 9 \* \***(wait for beep)

**\* 9 \* 1 \* \* 200 \* number1 \* number2 \* \***  (wait for beep -  read note 1)   

**\* 5 \* 1 \* 0030 \*** (wait for beep -- waits 3 seconds delay between number1 and number2 -- read note 2 below)    
**\* \* 235 \*** (wait for beep)

**\* \* 27 # \*** (waif for beep -- allows loud dialing tone, optional)

Hangup

note 1: As "number1", enter the first set of numbers (1-10) that you wish to dial before the pause.  Then, as "number2" enter the second set of numbers (1-10).  
  
note 2: Instead of 0030, you can use 0020 for a 2 second delay, or 0010 for a 1 second delay.
