---
title: "How do I set a blocked phone number to be an allowed phone number?"
sidebar_position: 11
format: md
---

Simply login to your account at <https://ultimatecallblocker.com/login> using any internet capable device (PC/MAC, tablet or cell phone). 

The easiest way to make sure you can receive calls from a particular phone number is simply by you adding the number or the name manually to your Whitelist by using the "Add Entry" section:

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/13169397063/original/YYEhjOHvyCkQQ2vJE4dXHtXW839A-ej6TA.png?1726894895)Important:  All the entries you add to your Whitelist will always overwrite (or supersede)  the entries in your BlackList or the Global BlackList.

Below are other ways to unblock a number:  
  
If the number is being blocked, this number should be in the user BlackList or in our Global Blacklist. So, first search your User BlackList in the section called "Current Data", and search the number you want to allow from the BlackList.  Next, select it with your mouse, and click the button below called "Move to WhiteList"  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/13169397010/original/Byy1viL2j3MsGzW_XEm2pMO24loeAf8ueg.png?1726894555)

If the number is not in your user BlackList, then type the number in the Global Backlist Section, and press "Search".  
![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/13169397043/original/IIXSz3rt26ia3r8-Dib7yoE7uUow85rjAg.png?1726894740)
