These errors occurred due to a number of reasons but the most common one is related to compatibility with browsers between our secured server and your device(s).  Below are some suggestions in case you get "“500 internal server error” when trying to login to <https://ultimatecallbocker.com>.  
  
1) Try using a different browser on your device e.g. try chrome, firefox, Edge, Internet Explorer, Opera, Safary, etc.    
  
2) If you tried different browsers on your device (laptop, desktop PC or tablet) and still getting these errors, try a different device e.g. tablet, laptop, desktop, etc. 

3) If you tried a different device and multiple browsers, then try connecting using a different Internet provider.

If none of this works, please contact us for further troubleshooting steps.
