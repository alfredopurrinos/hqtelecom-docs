title: "How to program to auto dial a number when taken off-hook?"
sidebar_position: 3
format: md
---

Step 1: \*\*\*8888 Wait for “BEEP” sound, log in the system.

Step 2: \*01\*1\*number\*\* Wait for “BEEP” sound, set access number.

Step 3: hang up.  
  
Where the number could be any string of digits from 1 to 16 digits long, including #.
