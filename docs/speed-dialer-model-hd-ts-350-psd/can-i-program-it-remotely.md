---
title: "Can I program it remotely?"
sidebar_position: 1
format: md
---

Yes, you can program this dialer locally or remotely.  If you are experiencing any issues while programming it locally, you should try using a phone line from a different PBX or phone service provider (Tip: consider programming it on a different location).  

You may also be able to program it remotely by calling it from another phone e.g. your cell phone.  Once you answer the call with the handset connected on the dialer, you should be able to input the programming codes as described in the [manual](https://www.hqtelecom.com/Portable-SpeedDialer-Manual.pdf).  This works well if there is no compatibility issues among the phone lines from the caller to the receiver.  

For more details on this [speed dialer](https://www.hqtelecom.com/dialers/speeddialer) product, please click [here.](https://www.hqtelecom.com/dialers/speeddialer)  If you still need help, please call our engineers at 786-221-5997.
